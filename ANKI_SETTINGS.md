# Recommended Anki settings — Japanese Grammar deck

## Presets

The deck ships with two options presets:

* **Japanese Grammar** — main preset (10 new cards/day, FSRS, sibling burying)
  bound to Module 00 only on a fresh import.
* **Japanese Grammar (opt-in)** — set to **0 new cards/day**, bound to every
  other module. You unlock a module by switching its preset to the main one.

## FSRS

Use **FSRS** (Anki ≥ 23.10). Recommended starting parameters:

* Desired retention: 0.90
* Maximum interval: 365 days (raise once you trust the deck)
* Easy bonus: 1.30
* Hard interval: 1.20
* Bury siblings: ON (recognition + production for the same point should not
  appear on the same day)

## Card display

* **Show audio button on front** — yes (you want to test listening)
* **Hide reading on front** — yes (furigana / romaji only on back)
* **Auto-play audio on flip** — recommended ON for the first 6 months,
  then optional

## Study path

| Phase | Weeks | Enable | Why |
|---|---|---|---|
| **1. Foundation** | 1–4 | 00 | Kana, particles, copula. Everything else presupposes this. |
| **2. N5 grammar** | 5–16 | 01 | Core morphology (te-form, -ます, -たい, conditionals start) |
| **3. N4 grammar** | 17–28 | 02 | Passive, causative, giving/receiving, evidentials |
| **4. N3 grammar** | 29–44 | 03 | The watershed — half of "intermediate Japanese" lives here |
| **5. N2 grammar** | 45–60 | 04 | News + business prose unlock |
| **6. N1 grammar** | 61–80 | 05 | Literary register, abstract relational nouns |
| **7. Keigo** | any time after wave-2 | 06 | Independent track — start as soon as you hit polite forms |
| **8. Casual + slang** | any time after wave-3 | 07 + 08 | For consuming anime / Twitter / podcasts |
| **9. SFP + onomatopoeia** | any time | 09 + 10 | Pragmatics + descriptive vocabulary |
| **10. Beyond N1** | C1+ | 12 | Idioms, 四字熟語, business set phrases |
| **L1 (anytime)** | any | 13 | Drill the contrasts that *your* L1 trips on |

Module 11 (Classical carryover) is recommended only when you start reading
news/literature.
