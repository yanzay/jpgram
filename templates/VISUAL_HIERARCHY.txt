DARK MODE BEHAVIOR

Automatic Detection:
  1. System preference: @media (prefers-color-scheme: dark)
  2. Anki's dark mode: body.night_mode class

Color Adaptation:
  All CSS variables switch automatically:
  --bg, --fg, --accent, --warning, --success, --muted

Box Shadows:
  Dark mode: lighter, lower contrast shadows (opacity adjustments)
  Prevents harsh glows on dark backgrounds

Backgrounds:
  Light gray muted backgrounds → darker grays with opacity
  Maintains readability and WCAG AA contrast

No Manual Toggle:
  Users don't need to do anything. Dark mode "just works"
  compatible with AnkiMobile, AnkiDroid, AnkiWeb
